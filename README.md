# pseudos [![NPM version](https://badge.fury.io/js/pseudos.svg)](http://badge.fury.io/js/pseudos)

> List of CSS pseudo-selectors.

## Install with [npm](npmjs.org)

```bash
npm i pseudos --save
```

## Run tests

```bash
npm test
```

## Usage

```js
var pseudos = require('pseudos');
//=> ['after', 'before', ...]
```
